# Grunt BiaPHP

this plugin from the possibility of using the BiaPHP

---

## How to install

```
npm install grunt-biaphp --save-dev

```


## How to use

In Gruntfile:


```
grunt.initConfig({

    biaphp: {

        compile: {
            cwd: '../path/to/bia'
            dest: '../path/to/php'

        }

    }

});


grunt.loadNpmTasks('grunt-biaphp');

```
